Hello {{name}}!

This is a dynamically generated webpage. You can see that your name was put into the file. Now we're getting somewhere. This can be linked to Python variables!
These variables are inserted using the Jinja2 template engine. The characters {{}} are used to insert expressions (and thus variables). {##} are used for comments, # for line statements, and {%%} for statements i.e. assignments (a = 1, b = test(), etc.) That means you can use code INSIDE a webpage template!
